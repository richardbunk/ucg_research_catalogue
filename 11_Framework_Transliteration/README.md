# 11_Framework_Transliteration

This directory contains planning and reconstruction documents governing the progressive transliteration of the existing Unified Composition Guide into the next-generation layered authority architecture.

The current UCG workbook and master indices are treated as source and planning artifacts.

Operational deployment progressively resolves into modular:

- Markdown (`.md`) human-readable doctrine
- YAML (`.yaml`) structured metadata and configuration
- JSON (`.json`) machine-readable indices

Framework transliteration is not direct migration. It is editorial reconstruction of institutional knowledge into governed, modular framework assets.
