# UCG Research Catalogue

This repository is the working research catalogue for the UCG revision project.

Its purpose is to collect, extract, catalogue, evaluate, synthesize, and reassemble UCG concepts from conversation transcripts, source documents, and existing UCG framework artifacts into a revised governed framework version.

This repo is intentionally staged:

1. Source corpus intake
2. Extraction candidate production
3. Catalogue indexing and tagging
4. Synthesis, conflict review, and canon selection
5. Framework reassembly into UCG vNext
6. Publication and reference packaging

Formal charters, extraction standards, templates, and governance rules will be added after the scaffold is reviewed.
