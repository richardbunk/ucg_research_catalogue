# UCG Research Catalogue

This repository is the working research catalogue and framework reconstruction environment for the next-generation Unified Composition Guide (UCG).

Its purpose is to collect, evaluate, extract, synthesize, organize, and transliterate institutional knowledge into a modular, governed framework architecture.

The repository supports two linked horizons:

1. **Research Ingestion Readiness** — accept external sources, transcripts, and framework sources into a consistent editorial pipeline.
2. **Framework Transliteration** — reconstruct the current operational UCG into modular `.md`, `.yaml`, and `.json` assets organized by layered authority.

## Active Architecture

- `00_Governance` — canonical RCG governance documents.
- `01_Source_Corpus` — source materials and acquired research assets.
- `02_Extraction_Workbench` — working extraction space.
- `03_Catalogue_Index` — catalogue indices and navigation aids.
- `04_Synthesis_Review` — synthesis and review materials.
- `05_Framework_Reassembly` — candidate framework reconstruction work.
- `06_Publication_Packages` — publication-ready packages and release material.
- `07_Process_Review` — review notes and process improvement records.
- `10_Operational_Protocols` — RIP workflow protocols.
- `11_Framework_Transliteration` — FTP planning and reconstruction documents.
- `90_Templates` — operational templates for source, research, extraction, knowledge, assessment, and review artifacts.
- `99_Archive` — retired or superseded material retained for traceability.

## Operating Principle

The Research Catalogue preserves institutional knowledge. The Unified Composition Guide implements institutional knowledge.

Knowledge accumulation is encouraged. Framework change remains deliberate, evidence-based, and architecturally reviewed.
