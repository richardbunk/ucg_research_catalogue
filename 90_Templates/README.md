# 90_Templates

Reusable operational templates for the UCG Research Catalogue.

Templates are not constitutional governance. They are working artifacts used to execute the governed editorial pipeline.

## Template Set

- `SR-001-source-record.md` — registers research and framework sources.
- `RS-001-research-session-record.md` — captures source understanding before extraction.
- `ER-001-extraction-record.md` — records reusable observations.
- `KA-001-knowledge-article.md` — preserves institutional knowledge.
- `KT-001-knowledge-tree-definition.md` — defines persistent knowledge domains.
- `FIA-001-framework-integration-assessment.md` — assesses whether knowledge should influence the framework.
- `FC-001-framework-candidate.md` — proposes framework implementation.
- `ERV-001-editorial-review-record.md` — validates proposed implementation before canonical adoption.

## Operating Principle

Templates guide editorial judgment. They are not merely administrative forms.
