# 00_Governance

Canonical governance for the UCG Research Catalogue.

This directory contains the RCG series: foundational governance documents that define the purpose, process, architecture, editorial standards, publication authority, and validation principles of the Research Catalogue.

Operational protocols, planning documents, and working templates are stored outside this directory.

## Active Governance Set

- `RCG-001_Research_Catalogue_Charter.md` — Research Catalogue charter.
- `RCG-002-research-extraction-and-framework-codification-protocol.md` — extraction and codification governance.
- `RCG-003-research-asset-and-information-architecture-standard.md` — information architecture standard.
- `RCG-004-research-editorial-standards.md` — editorial standards.
- `RCG-005-canonical-publication-and-version-governance-standard.md` — canonical publication and version governance.
- `RCG-006-research-quality-assurance-and-validation-protocol.md` — quality assurance and validation protocol.

## Related Directories

- `10_Operational_Protocols` — RIP operational workflows.
- `11_Framework_Transliteration` — FTP transliteration and reconstruction planning.
- `90_Templates` — reusable operational artifact templates.
- `99_Archive` — retired governance retained for traceability.

## Governance Principle

Governance defines enduring authority. Operational protocols define practice. Templates support execution.
