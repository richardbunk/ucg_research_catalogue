# 10_Operational_Protocols

Operational protocols define day-to-day workflows used to run the UCG Research Catalogue.

This directory is distinct from `00_Governance`.

- Governance documents define enduring authority.
- Operational protocols define repeatable practice.

Expected protocol families include:

- RIP — Research Ingestion Protocols
- Publication workflows
- Editorial review workflows
- Research processing workflows

Operational protocols remain governed by the RCG series but may evolve as practical use reveals better methods.
